## 실행 방법
1. Visual Studio로 `main.c` 열기
2. F5 또는 Ctrl+F5로 실행
3. 콘솔에서 키 입력

### 🔁 게임 루프 구성
- 상태 출력
- 기분 나빠짐 (확률적)
- 기분에 따라 이동 & 행동
- 방 상태 출력
- 상호작용 선택
- CP 생산
- 상점에서 물건 구매
- (3턴) 돌발 퀘스트 발생


## 🧠 돌발퀘스트 (3턴)
- 미니게임: 1~5 숫자 맞추기
- 정답 시 보상, 실패 시 야옹이 삐짐
